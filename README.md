## About

Mavlina is a powerful [Node.js](https://nodejs.org) module that allows you to easily interact with the
[Mavlina's API](https://discord.com/developers/docs/intro).

- Easy to use
- Lightweight
- Object-oriented
- Performant

## Installation

**Node.js 16.9.0 or newer is required.**

```sh-session
npm install mavlina
yarn add mavlina
pnpm add mavlina
```
